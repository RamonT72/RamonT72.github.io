Glitch Collage

Tool(s): Text Edit, Adobe Photoshop

The image I chose for this project was a character from one of my favorite anime, One Piece. To create this image first I used Text edit to alter the original image's code and give it a glitch effect. After I found an effect I liked I went onto Photoshop to finish it. I sliced the image vertically and the horizontally. I ended up with a checkerboard like design.
